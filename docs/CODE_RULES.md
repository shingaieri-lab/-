# コード品質ルール

## 重複禁止

- 同じロジック・定数・テンプレートが2か所以上に存在してはいけない
- 共通関数またはグローバル変数に統一する
- 特に重複しやすいもの：祝日リスト、メールテンプレート初期値、OAuthエラーハンドリング

## React 固有

- useEffect の依存配列に ref.current を入れない
  - ❌ useEffect(() => { ... }, [ref.current])
  - ✅ useEffect(() => { ... }, [])

## パスワード保存

- bcrypt ハッシュのみ保存。復号可能な暗号化（AES等）との二重保存禁止

## 外部API依存

- 外部API（祝日API等）が失敗した場合はローカルのフォールバックデータを使う
